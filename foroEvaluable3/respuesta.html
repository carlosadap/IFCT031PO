Para crear un nuevo objeto en JavaScript, podemos hacerlo de diferentes maneras, una de ellas es con el siguiente código:

  
    const objeto = { 
      propiedad 1 : valor_1 , // propiedad_# puede ser un identificador...
      2 : valor_2             // o un número... 
      "propiedad n": valor_n; // o una cadena
    }

Así, para crear el objeto "persona" con las propiedades requeridas: 

  
    const persona = {
      Nombre: "Francisco",
      Apellidos: "Prieto Garcia",
      Edad: 43,
    }

Para mostrar en pantalla los datos de un otro archivo, primero tenemos que conectar ambos archivos, como por ejemplo utilizando el comando <script>

  
    <script type="text/javascript" src="archivodelobjeto.js"></script>

Luego, una manera para que el objeto aparezca en la pantalla es convertirlo en JSON y entonces añadirlo al elemento de interés del HTML por intermedio del método innerHTML.

  
    <script> document.querySelector(".obExterno").innerHTML = JSON.stringify(persona); </script>
