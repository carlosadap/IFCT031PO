• El elemento HTML <html> (o elemento HTML raiz) representa la raiz de un documento HTML. El resto de elementos descienden de este elemento.
  • El elemento HTML <head> provee información general (metadatos) acerda del documento, incluyendo su título y enlaces a scripts y hojas de estilos.
  • El elemento HTML <meta> representa metadatos que no pueden ser representados por otro elemento relacionado a metados.
  • El elemento HTML <title> indica el título del documento. El contenido de title suele ser mostrado en la barra de título de la ventana del navegador.
  • El elemento HTML <body> representa el contenido de um documento HTML. Solo puede haber un elemento <body> en un documento.
  • Los elementos de encabezado implementan seis niveles de encabezado del documento, <h1> es el más importante, y <h6>, el menos importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. La información de encabezado puede ser usada por los agentes usuarios, por ejemplo, para construir una tabla de contenidos para un documento automáticamente.
  • La etiqueta <meta> es un elemento de cabecera, por lo tanto, para utilizarla la incluimos en el elemento <head>.
  • Tiene como atributos <name>, <http-equiv>, <charset> y <itemprop>
  • La etiqueta palabras clave contiene palabras relevantes para el contenido de la página, separada por comas. Se supone que estas palabras claves son utilizadas por los motores de búsqueda para determinar la relevancia de la página en términos de búsqueda.
  • Ejemplo de utilización: 
    
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
